# Slarof Additions
- Дополнительный текстурпак Minecraft сервера Slarof, расширяющий контент

### Автоматическое обновление текстурпака
1. Установите модификацию [Resclone](https://modrinth.com/mod/resclone/versions) в `.minecraft/mods`
2. Вставьте [конфигурацию](https://github.com/SlarofMC/Slarof_Additions/blob/resourcepacks/.minecraft/config/resclone/config.json) в `.minecraft/config`
3. Готово
